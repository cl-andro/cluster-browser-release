# Cluster Browser - Release Repository

This repository contains the build workflow scripts, license, and instructions to download and install **Cluster Browser** — a privacy-focused Chromium-based browser with native terminal integration.

---

## 📥 How to Download & Install

1. Navigate to the **Releases** section on the right side of this GitHub repository page.
2. Download the latest `cluster-browser-release.apk`.
3. Transfer the APK to your phone and install it (make sure you allow installation from "Unknown Sources" in your browser/file explorer if prompted).

---

## 📦 About Cluster Browser

**Cluster Browser** is a fork of Cromite with custom UI enhancements, dynamic dark/light theme options, a native terminal overlay backed by the cl-andro service, and customized bookmarks.

### Key Features
- Chromium-based engine for high-performance and secure browsing
- Built-in E2EE chat overlay (`Cl-chat`) and native terminal integration
- Enhanced Read Mode parsing using the browser's native DOM distiller
- Custom bookmarks and profile manager integration
